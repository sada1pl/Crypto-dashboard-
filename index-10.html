⚙️ Konfiguracja Synchronizacji

Dane szyfrowane są AES-256 przed wysłaniem — serwer nigdy nie widzi Twoich kluczy API ani haseł. Tylko Ty masz klucz szyfrowania.

1
Wejdź na dash.cloudflare.com → Workers & Pages → Create Worker → wklej kod z pliku worker.js (pobierz poniżej)
2
Workers → KV → Create namespace o nazwie CRYPTO_SYNC → przypisz do workera jako binding KV
3
Skopiuj URL workera (np. https://crypto-sync.twoja-nazwa.workers.dev) i wklej poniżej

Pobierz zaszyfrowany plik .json z wszystkimi danymi. Wgraj go na innym urządzeniu aby przywrócić.

⚠️ Bez tego klucza nie odszyfruje danych na innym urządzeniu. Zapisz go osobno!
